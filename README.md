# 🌍 Trip — Travel Booking Platform

A modern travel booking web application built with Next.js.

Users can explore destinations, plan trips, and manage bookings through a fast and responsive interface.

---

## 🚀 Tech Stack

### Frontend
- Next.js
- React
- JavaScript
- CSS / Tailwind (if using)

### Backend
- Node.js
- Express.js (if using)

### Database
- MongoDB (if using)

---

## 📂 Project Structure

```bash
Trip/
│
├── client/        # Frontend application
├── server/        # Backend API
```

---

## ✨ Features

- Responsive travel booking UI
- Destination browsing
- Trip management
- Authentication system
- Booking workflow
- API integration

---

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Soujanya26sb/Trip.git
```

### Go to project directory

```bash
cd Trip
```

---

# Client Setup

```bash
cd client
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:3000
```

---

# Server Setup

```bash
cd server
npm install
npm run dev
```
